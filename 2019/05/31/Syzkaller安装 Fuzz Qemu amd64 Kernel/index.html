Syzkaller安装 Fuzz Qemu amd64 Kernel

syzkaller官网上有介绍如何在Ubuntu宿主机上用qemu方法fuzz x86_64的Linux内核，但是步骤很分散，在好几个页面上，而且还可能有一些坑，后面会讲到。
首先介绍一下我的环境：

  • Ubuntu 16.04 x86_64
  • gcc 8.2.0
  • linux-5.1
  • go1.12.5
